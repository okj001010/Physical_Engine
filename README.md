# Physical_Engine

### [2017년도 2학기 대구과학고등학교 프로그래밍 실습2 프로젝트]

역학 상황 시뮬레이션 (시간, 위치, 속도, 가속도)

[물체], [장애물] 객체를 활용하여 자유로운 물리 환경 구성

[물체]에 위치, 속도, 가속도 값을 입력하면 뉴턴 역학에 따라 시뮬레이션 진행

시간에 따른 위치, 속도, 가속도 그래프 출력
